## Tips

1. Always upload character reference images — consistency depends on it
2. Generate 3-4 variations per scene, pick the one that looks most "photographic"
3. If results look too "CG": add more "real person, photograph, DSLR" keywords
4. If skin looks too smooth: emphasize "visible pores, skin texture, no airbrushing"
5. If lighting is too dramatic: add "soft fill light on face, natural lighting"
6. Save winners as: `scene_01.png` through `scene_08.png`
