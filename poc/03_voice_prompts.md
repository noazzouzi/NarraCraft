# Voice Prompts — ElevenLabs

## Voice Setup

**Character:** Jill Valentine
**Voice style:** Young woman, late 20s. Confident, military-trained, but human — not robotic. Think "soldier telling a story around a campfire." Slight edge of anger when talking about Umbrella, softens slightly when showing empathy for Wesker.

**ElevenLabs settings:**
- Pick a female voice that sounds: confident, clear, mid-range pitch, slight rasp
- Recommended voices to try: "Rachel", "Bella", or browse for a voice that fits
- Stability: ~0.50 (some expressiveness)
- Similarity boost: ~0.75
- Style exaggeration: ~0.30

## Scene-by-Scene Voice Lines

### Scene 1 (6 sec)
**Direction:** Intense, grabbing attention, punchy delivery. Emphasis on "never had a choice."
```
Did you know? Albert Wesker — the man who betrayed us all — never had a choice.
```

### Scene 2 (9 sec)
**Direction:** Slower, more somber. Setting up the backstory. "Kidnapped" should hit hard.
```
Before he was our captain, before STARS, he was a child — one of dozens kidnapped by Umbrella's founder, Oswell Spencer.
```

### Scene 3 (8 sec)
**Direction:** Disgust creeping in. "Injected, tested, conditioned" — each word deliberate, almost spat out.
```
Spencer called it Project W. A eugenics program to create the perfect human. They were injected, tested, conditioned.
```

### Scene 4 (10 sec)
**Direction:** Building intensity. "Only one survived" — pause slightly before "Wesker." Then build momentum through the list. Awe mixed with horror.
```
Out of all those children, only one survived — Wesker. His superhuman speed, his strength, his obsession with evolution — all of it was programmed into him.
```

### Scene 5 (8 sec)
**Direction:** Quiet weight. Almost sympathetic at first, then voice hardens on "He was built to be one." Final line is the mic drop — deliver it with conviction.
```
The man Chris and I spent years fighting? He was Spencer's weapon from the start. Wesker never chose to be a monster. He was built to be one.
```

## Tips

1. Generate each scene as a separate audio clip — easier to time in CapCut
2. Listen to each clip — if the pacing feels off, regenerate just that scene
3. Total audio should be ~41-47 seconds (leaving room for pauses and transitions)
4. Save files as: `scene_01.mp3`, `scene_02.mp3`, etc.
