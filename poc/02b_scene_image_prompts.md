1. Generate 2-3 variations per scene, pick the best
2. If a character's face drifts from the reference, try regenerating — or slightly rephrase the description
3. Scene 2 (flashback) doesn't need character references — it's a unique scene
4. Scene 3 should be easy — it's just Jill close-up, similar to your narrator pose
5. Save approved scene images as: `scene_01.png`, `scene_02.png`, etc.
